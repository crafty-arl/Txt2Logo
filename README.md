# txt 2 logo generator by craft the future 🎨✨

hey everyone! welcome to the txt 2 logo generator, where you can create custom logos using ai. let's dive into the world of ai-generated logos and make something amazing together! 🚀

## setup 🛠️

1. clone this repository:
    ```sh
    git clone https://github.com/crafty-arl/Txt2Logo.git
    cd Txt2Logo
    ```

2. install the required packages:
    ```sh
    pip install -r requirements.txt
    ```

3. set up your streamlit secrets:
    create a file called `.streamlit/secrets.toml` and add your api keys:
    ```toml
    [secrets]
    MAGIC_API = "your_magic_api_key"
    OPENAI_API = "your_openai_api_key"
    ```

4. run the streamlit app:
    ```sh
    streamlit run app.py
    ```
